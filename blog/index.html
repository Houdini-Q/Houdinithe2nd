The journey so far has been a long one. I've learned to use percentages instead of being precise when placing elements within the webpage. Percentages allow those elements to scale no matter the size of the screen it is being displayed on, while being more direct with pixels or "px" will cause your elements to become unaligned when the screen size changes. Maybe this little tid-bit of information will help readers and developers alike.
